Install
------------------------------------------------------------------------------

``bgs_deploy`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install bgs_deploy

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade bgs_deploy
